Synopsis:

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 Cities.

After Assembling the data set, we used matplotlib to plot various aspects of the weather vs. latitude. Factors we looked into included: temperature, cloudiness, wind speed, and humidity. This site provides a detailed analysis of each plot, as well as a comparison of the various aspects.
